## 📌 Descripción

**NetflixBasicNavigation** es un proyecto académico que enseña los conceptos fundamentales de navegación en Android con **Jetpack Compose Navigation**.

A través de una interfaz inspirada en Netflix, comprenderás cómo estructurar pantallas, gestionar rutas y pasar datos entre composables de manera clara y práctica.

---

## 🚀 Objetivos de aprendizaje

| # | Objetivo |
|---|----------|
| 1 | Comprender la navegación en Jetpack Compose |
| 2 | Implementar `NavHost` y `NavController` |
| 3 | Definir y gestionar rutas |
| 4 | Navegar entre múltiples pantallas |

---

| Tecnología | Descripción |

## ▶️ Cómo ejecutar el proyecto

**1. Hacer un fork del repositorio**

- Ir al repositorio original en GitHub
- Hacer clic en el botón **Fork** (arriba a la derecha)

**2. Clonar tu fork**

```bash
git clone https://github.com/tu-usuario/NetflixBasicNavigation.git
```

**3. Abrir en Android Studio y ejecutar** ▶️
